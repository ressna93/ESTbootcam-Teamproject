# 메인 페이지 (Main Page)

## 개요
ZADU 오픈 마켓의 메인 홈페이지로, 상품을 탐색하고 검색할 수 있는 핵심 기능을 제공합니다.

## 주요 기능

### 1. 히어로 슬라이더 (Hero Slider)
- **Swiper 라이브러리**: 캐러셀/슬라이드 기능을 구현합니다
- **자동 재생**: 3초마다 자동으로 슬라이드가 전환됩니다
- **무한 루프**: 마지막 슬라이드 이후 첫 번째 슬라이드로 돌아갑니다
- **페이지네이션**: 하단에 현재 슬라이드를 나타내는 인디케이터를 표시합니다
- **네비게이션 버튼**: 좌우 화살표로 수동 제어가 가능합니다
- **터치 스와이프**: 모바일에서 손가락으로 스와이프할 수 있습니다
- **그라디언트 배경**: 보라색 계열의 그라디언트로 시각적 효과를 제공합니다

### 2. 탭 UI (Category Tabs)
- **카테고리 분류**: 전체 상품, 액세서리, 문구류로 분류됩니다
- **활성화 표시**: 선택된 탭은 하단 테두리와 색상으로 강조됩니다
- **키보드 네비게이션**: Arrow 키, Home, End 키로 탭 이동이 가능합니다
- **접근성**: ARIA 속성(role, aria-selected, aria-controls)을 사용합니다
- **호버 효과**: 마우스 오버 시 색상 변경으로 피드백을 제공합니다

### 3. 상품 목록 표시
- **그리드 레이아웃**: 반응형 그리드로 상품 카드를 배치합니다
- **상품 정보**: 이미지, 판매자명, 상품명, 가격을 표시합니다
- **Lazy Loading**: 이미지를 지연 로딩하여 성능을 최적화합니다
- **호버 효과**: 마우스 오버 시 카드가 살짝 올라가고 그림자가 생깁니다
- **클릭 이동**: 상품 카드 클릭 시 상세 페이지로 이동합니다

### 4. 페이지네이션
- **페이지 단위**: 한 페이지당 8개의 상품을 표시합니다
- **페이지 번호**: 현재 페이지 기준 앞뒤 2페이지씩 표시합니다
- **생략 표시**: 중간 페이지는 "..."로 생략 표시합니다
- **이전/다음 버튼**: "‹", "›" 버튼으로 이동할 수 있습니다
- **자동 스크롤**: 페이지 변경 시 상단으로 부드럽게 스크롤됩니다
- **활성화 표시**: 현재 페이지는 녹색 배경으로 강조됩니다

### 5. 검색 기능
- **검색 폼**: 헤더의 검색 폼과 연동됩니다
- **검색 API**: 검색어를 API로 전송하여 결과를 가져옵니다
- **빈 검색어 처리**: 검색어가 없으면 전체 상품을 표시합니다
- **로딩 상태**: 검색 중 스켈레톤 UI를 표시합니다
- **에러 처리**: 검색 실패 시 콘솔에 에러를 출력합니다

### 6. 로딩 및 에러 처리
- **로딩 스켈레톤**: 데이터 로드 중 회색 애니메이션 카드를 표시합니다
- **스켈레톤 애니메이션**: 투명도가 변화하는 펄스 효과를 적용합니다
- **에러 메시지**: API 호출 실패 시 에러 메시지와 재시도 버튼을 표시합니다
- **재시도 기능**: "다시 시도" 버튼으로 상품 로드를 재시도할 수 있습니다

### 7. 반응형 디자인
#### 모바일 (max-width: 480px)
- 슬라이더 높이: 288px (18rem)
- 그리드: 2열 배치
- 네비게이션 버튼 숨김
- 탭 가로 스크롤 지원

#### 태블릿 (481px ~ 1024px)
- 슬라이더 높이: 320px (20rem)
- 그리드: 3열 배치
- 폰트 크기 조정

#### 데스크톱 (1025px 이상)
- 슬라이더 높이: 400px (25rem)
- 그리드: 4열 배치
- 최대 너비: 1200px (75rem)

## 사용 기술
- **Vanilla JavaScript (ES6+)**: 모듈 시스템, async/await 사용
- **Swiper.js**: 슬라이더/캐러셀 라이브러리 (v11)
- **CSS Grid**: 반응형 레이아웃
- **CSS Custom Properties**: CSS 변수로 스타일 관리
- **Fetch API**: REST API 통신
- **Lazy Loading**: 이미지 최적화

## API 연동
| 함수 | 엔드포인트 | 설명 |
|------|-----------|------|
| `API.getProducts()` | `GET /products/` | 전체 상품 목록 조회 |
| `API.searchProducts(query)` | `GET /products/?search={query}` | 상품 검색 |

## 주요 함수

### 초기화 함수
- `initSwiper()`: Swiper 슬라이더를 초기화합니다
- `initTabs()`: 탭 UI 이벤트 리스너를 설정합니다
- `initSearch()`: 검색 기능을 초기화합니다

### 데이터 관리
- `loadProducts()`: API에서 상품 데이터를 가져옵니다
- `renderProducts()`: 필터링 및 페이지네이션을 적용하여 상품을 렌더링합니다

### UI 렌더링
- `renderPagination(totalItems)`: 페이지네이션 버튼을 생성합니다
- `switchTab(selectedButton)`: 탭을 전환하고 카테고리를 변경합니다

### 전역 변수
- `allProducts`: 전체 상품 배열을 저장합니다
- `currentPage`: 현재 페이지 번호 (기본값: 1)
- `ITEMS_PER_PAGE`: 페이지당 상품 수 (기본값: 8)
- `currentCategory`: 현재 선택된 카테고리 (기본값: 'all')

## CSS 주요 스타일

### 색상 체계
- **주요 색상**: #21bf48 (녹색 - 브랜드 컬러)
- **그라디언트**: #667eea → #764ba2 (보라색 계열)
- **텍스트**: #333 (기본), #666 (부가 정보), #999 (연한 회색)
- **배경**: #fff (흰색), #f8f8f8 (연한 회색)

### 레이아웃
- **컨테이너 최대 너비**: 1200px (75rem)
- **그리드 갭**: 24px (1.5rem)
- **카드 테두리**: 1px solid #e5e5e5
- **카드 모서리**: 8px (0.5rem) 둥근 모서리

### 애니메이션
- **호버 효과**: 0.3초 ease 전환
- **카드 리프트**: translateY(-5px) 이동
- **스켈레톤**: 1.5초 무한 반복 페이드 애니메이션

### 타이포그래피
- **기본 폰트**: Spoqa Han Sans Neo, Apple SD Gothic Neo, Noto Sans KR
- **라인 높이**: 1.6
- **슬라이더 제목**: 40px (2.5rem), 굵게
- **상품명**: 16px (1rem), 중간 굵기
- **가격**: 18px (1.125rem), 굵게 (강조)

## 접근성 (Accessibility)
- **ARIA 속성**: role, aria-selected, aria-controls, aria-label, aria-current 사용
- **키보드 네비게이션**: Arrow 키, Home, End 키로 탭 이동 가능
- **포커스 표시**: outline으로 포커스 상태를 명확히 표시
- **대체 텍스트**: 모든 이미지에 alt 속성 제공
- **시맨틱 HTML**: section, article, button 등 의미있는 태그 사용

## 성능 최적화
- **Lazy Loading**: 이미지 지연 로딩으로 초기 로딩 속도 개선
- **비동기 처리**: async/await로 API 호출을 비동기로 처리
- **스켈레톤 UI**: 로딩 중 시각적 피드백 제공
- **모듈 시스템**: ES6 모듈로 코드를 분리하여 유지보수성 향상
- **CSS 최적화**: 수평 스크롤 방지, max-width 제한

## 사용자 경험 (UX)
- **부드러운 스크롤**: 페이지 이동 시 smooth scroll 효과
- **시각적 피드백**: 호버, 클릭, 활성화 상태를 명확히 표시
- **로딩 상태 표시**: 스켈레톤 UI로 데이터 로딩 중임을 알림
- **에러 복구**: 재시도 버튼으로 사용자가 직접 복구 가능
- **터치 친화적**: 모바일에서 스와이프 및 터치 인터랙션 지원

## 확장 가능성
- **카테고리 필터링**: 현재는 전체 상품만 표시하지만, 카테고리별 필터링 로직을 쉽게 추가할 수 있습니다
- **정렬 기능**: 가격순, 인기순 등의 정렬 기능을 추가할 수 있습니다
- **무한 스크롤**: 페이지네이션 대신 무한 스크롤로 변경 가능합니다
- **찜하기 기능**: 상품 카드에 찜하기 버튼을 추가할 수 있습니다
- **최근 본 상품**: localStorage를 활용하여 최근 본 상품을 추적할 수 있습니다
